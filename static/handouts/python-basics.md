## 重點提示

Python 是一個 ==簡單易學== 的程式設計語言，特別適合 ==blue:初學者== 入門。

重要概念：==green:變數命名規則== 必須遵守，==orange:縮排== 在 Python 中非常重要！

使用 ==purple:函數== 和 ==pink:類別== 來組織程式碼。

## 基本語法

### 變數宣告

在 Python 中，==orange:不需要明確宣告變數類型==，直接賦值即可：

```python
name = "學生"
age = 16
height = 165.5
is_student = True
```

### 資料類型

Python 主要的資料類型包括：
- ==blue:整數 (int)==
- ==blue:浮點數 (float)==
- ==green:字串 (str)==
- ==purple:布林值 (bool)==
- ==pink:列表 (list)==
- ==orange:字典 (dict)==

## 範例程式碼

```python
# Hello World 程式
print("Hello, World!")

# 變數宣告
name = "學生"
age = 16

print(f"我是{name}，今年{age}歲")

# 列表操作
fruits = ["蘋果", "香蕉", "橘子"]
for fruit in fruits:
    print(fruit)
```

## 重要提醒

記住：Python 使用 ==縮排來定義程式碼區塊==，而不是大括號！

==yellow:注意縮排的一致性== 是寫好 Python 程式的關鍵！
